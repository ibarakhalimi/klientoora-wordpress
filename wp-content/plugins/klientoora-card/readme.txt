A WordPress plugin scaffold for Klientoora card features.

== Description ==

Klientoora Card provides a clean plugin foundation built with WordPress Coding Standards in mind.

== Installation ==

1. Upload the `klientoora-card` directory to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` screen in WordPress.

== Frequently Asked Questions ==

= Does this plugin include business logic yet? =

No. This is a standards-focused scaffold ready for feature development.
